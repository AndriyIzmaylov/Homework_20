To run UI tests execute in command line from the root of the project folder:
python "features UI\runner_ui.py"

To run API tests execute in command line from the root of the project folder:
python "features API\runner_api.py"

To run API and UI tests execute in command line from the root of the project folder:
python "features UI\runner_ui.py" "features API\runner_api.py"
